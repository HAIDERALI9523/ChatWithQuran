The "Name" column in the qurantranslation database contains the names of Surahs (chapters) in the Quran stored as text The values are repeated to represent individual verses or specific details within each Surah This column is used to group and identify data facilitating queries and processing,

The "Surah" column in the qurantranslation database contains numerical identifiers (integers) for the Surahs in the Quran These values are repeated to represent individual verses or related details within a Surah allowing for the unique identification and referencing of Surahs in structured queries, grouping, and data organization,

The "Ayat" column in the qurantranslation database contains numerical identifiers (integers) representing individual verses (Ayats) within different Surahs The values correspond to the sequential position of each Ayat within its respective Surah and the column is used to uniquely reference and retrieve verse-level data for structured queries,

The "Arabic" column in the qurantranslation database contains the original Arabic text of the Quranic verses (Ayats) preserving the authenticity of the Quranic script Each value corresponds to a specific verse in its respective Surah This column is used to enable linguistic processing and to support queries and retrieval of verses in their original form for analysis and display,

The "NumberOfVerses" column in the qurantranslation database contains the total number of verses in each Surah of the Quran For example the "NumberOfVerses" for Surah Al-Fatihah is 7 and for Surah Al-Baqarah it is 286 This column helps track the length of each Surah assisting users in understanding the scope and structure of the Surah when querying specific information The system returns relevant content based on the number of verses in a Surah supporting more targeted searches related to the verses and themes of the Surah,

The "NumberOfRukus" column in the qurantranslation database contains the total number of rukus (sections) in each Surah of the Quran For example Surah Al-Fatihah has 1 ruku and Surah Al-Baqarah has 40 rukus This column categorizes the Surahs into sections for easier reference and analysis The system can filter and return relevant data based on the number of rukus providing a structured approach to accessing the content of the Surahs,
